## 4. Technical Notes

### 4.1 Key Design Decisions

**Accordion Behavior**
- Only one DocTabItem expanded at a time
- Expanding a tab also sets it as active (shows in editor)
- PromptPanel shows for all doc types when expanded

**Generate Flow**

| Doc Type | Input | Output |
|----------|-------|--------|
| Brief | Prompt + Attributes | Story framework â†’ `docs[type='brief']` |
| Draft | Prompt + Brief content | Full draft â†’ `docs[type='draft']` |
| Script | Prompt + Draft content | Scene script â†’ `docs[type='script']` |

**No Language Impact**
DocCreativeSpace is NOT affected by `currentLanguage`. Document content is single-language (original language only).

### 4.2 Layout Constants

| Element | Value |
|---------|-------|
| Sidebar width | 280px (fixed) |
| Sidebar min-width | 240px |
| Editor min-width | 400px |

| DocTabList | `tablist` | `aria-orientation="vertical"` |
